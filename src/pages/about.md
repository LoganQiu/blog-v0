# About

This website is built with Astro using a theme inspired by hugo-paged.

## The Theme

The hugo-paged theme is designed to create beautiful, print-ready documents while maintaining excellent web accessibility. It uses CSS that mimics the appearance of printed pages.

## Features

- Clean, minimal design focused on readability
- Support for blog posts with categories and tags
- Responsive layout that works on all devices
- Typography optimized for long-form reading

## Technologies

This Astro version uses:

- Astro for static site generation
- TypeScript for type safety
- Content Collections for managing posts
- Pure CSS for styling (no frameworks needed)
